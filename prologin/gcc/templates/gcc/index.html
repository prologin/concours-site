{% trans 'Prerequisites' %}

{% blocktrans %} The internship's goal is to adapt to each of you so you do not have to have a minimal level in IT before the internship. Beside, go take a look to {% endblocktrans %} {% trans 'some resources' %} {% blocktrans %} to peek a look at what's coming ! {% endblocktrans %}

{% trans 'Free internship' %}

{% blocktrans %} Internships are free and meals are provided. The association does not provide any accommodation or transportation to the stage place. But if you have any financial issues, send us a mail to {% endblocktrans %} info@prologin.org {% blocktrans %} and we will talk about a possible scholarship. {% endblocktrans %}

{% trans 'For those who cannot apply' %}

{% blocktrans %} The IT is not closed so far ! Do not hesitate to discover the recommended resources to start and if you are less than 21 years old, you can take part to the 27th edition {% endblocktrans %} {% trans 'of the Prologin contest' %}. {% blocktrans %} The first online qualification stage is now open. {% endblocktrans %}

{% trans 'The program' %}

{% blocktrans %} Internship last one week during which there is some practical courses on diverse subjects(Python introduction, networks, web), and some conferences where the women speakers work in IT. Finally, the week ends with a small project where creativity is inferred {% endblocktrans %}
